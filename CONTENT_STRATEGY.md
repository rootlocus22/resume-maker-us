## PART 5: TECHNICAL SEO REQUIREMENTS

### On-Page SEO Checklist (Every Page)
- [ ] Unique title tag (55-60 chars) with primary keyword
- [ ] Meta description (150-160 chars) with CTA
- [ ] H1 tag with primary keyword
- [ ] H2/H3 hierarchy with secondary keywords
- [ ] Image alt tags with keywords
- [ ] Internal links (5-10 per page)
- [ ] External links to authoritative sources (BLS, SHRM)
- [ ] Schema markup (FAQ, HowTo, Article, or Product)
- [ ] Canonical URL
- [ ] Open Graph tags
- [ ] Page speed < 3 seconds
- [ ] Mobile-first responsive design

### Site-Wide Technical SEO
- [ ] XML sitemap with all pages (update sitemap.js)
- [ ] robots.txt optimized
- [ ] SSL certificate (HTTPS)
- [ ] Core Web Vitals passing
- [ ] Structured data for breadcrumbs
- [ ] 301 redirects for any moved pages
- [ ] hreflang tags (if multi-language planned)
- [ ] Clean URL structure (no parameters)

### Internal Linking Strategy
Every page should link to:
1. Related resume examples (3-5 links)
2. Matching cover letter example (1 link)
3. Relevant blog post (2-3 links)
4. Tool page CTA (resume builder, ATS checker)
5. Template page (1 link)

## APPENDIX B: CONTENT QUALITY GUIDELINES

### What Makes a Top-Ranking Resume Example Page (Surfer SEO Analysis)

Based on analysis of top-ranking pages for "[job title] resume":

1. **Word Count:** 2,500-4,500 words (average of top 5 results)
2. **Headers:** 15-25 H2/H3 headers
3. **Images:** 3-5 images (resume screenshots, infographics)
4. **Internal Links:** 8-12 per page
5. **External Links:** 2-3 to authoritative sources
6. **FAQ Section:** 5-8 questions with schema markup
7. **NLP Keywords:** Include related terms naturally
   - For "nurse resume": patient care, clinical skills, healthcare, RN, BSN, medication administration, vital signs, EMR, charting
   - For "software engineer resume": programming languages, algorithms, code review, agile, CI/CD, microservices, APIs, testing
8. **Readability:** Flesch-Kincaid Grade 8-10
9. **Paragraphs:** Short (2-3 sentences max)
10. **Lists:** Use bullet points extensively

### Content Template Structure for Resume Example Pages:
```
1. Introduction (100-150 words)
2. [Job Title] Resume Example - Full Preview (screenshot + text)
3. How to Write a [Job Title] Resume (500-800 words)
```
